# NovaBlox MCP Server

## Install

```bash
cd /home/nova/NovaBlox/mcp-server
pip install -r requirements.txt
```

## Run

```bash
ROBLOXBRIDGE_HOST=localhost ROBLOXBRIDGE_PORT=30010 python novablox_mcp.py
```
